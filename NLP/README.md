# Natural Language Processing Lab Experiments

This repository contains implementations of **Natural Language Processing (NLP) laboratory experiments** performed as part of the Artificial Intelligence coursework.

All experiments are implemented in a **single Jupyter Notebook**:

`NLP_Lab_Experiments.ipynb`

The notebook includes theoretical explanations, implementation code, and results for each experiment.

---

## Experiments Included

### Experiment 1

**Aim:** Clean and preprocess text manually and using the NLTK library.

### Experiment 2

**Aim:** Represent text data as numerical features using

* Bag of Words (BoW)
* Term Frequency – Inverse Document Frequency (TF-IDF)

### Experiment 3

**Aim:** Perform **sentiment analysis** on movie reviews by preprocessing text, extracting features, and applying classification models.

### Experiment 4

**Aim:** Implement **Word2Vec models (CBOW and Skip-gram)** to generate word embeddings that capture semantic relationships between words.

### Experiment 5

**Aim:** Fine-tune the **BERT (Bidirectional Encoder Representations from Transformers)** model for movie review sentiment classification.

### Experiment 6

**Aim:** Implement **Word2Vec using NumPy** and perform classification using

* Support Vector Machine (SVM)
* Naive Bayes

### Experiment 7

**Aim:** Visualize word embeddings using **t-Distributed Stochastic Neighbor Embedding (t-SNE)**.
