## Areas Covered

### Java Programming

Implementation of fundamental programming concepts including:

* Sorting algorithms (Bubble Sort)
* Searching algorithms (Binary Search)
* Recursion (Factorial, Fibonacci)
* Array manipulation
* Matrix operations
* Duplicate removal
* Prime number checking

These assignments focus on **core algorithmic thinking and problem-solving using Java**.

---

### Natural Language Processing (NLP)

Contains multiple NLP experiments implemented in a single notebook covering:

* Text preprocessing
* Tokenization and stopword removal
* Bag of Words and TF-IDF
* Word embeddings (Word2Vec, GloVe)
* Sentiment analysis
* Deep learning models (RNN, LSTM)
* BERT fine-tuning
* Visualization using t-SNE

---

### Multimodal Learning

Experiments combining **text and image data** for multimodal machine learning tasks.

Concepts explored include:

* Feature extraction from images
* Text embeddings
* Multimodal data fusion
* Deep learning models for multimodal classification

### Conversational AI

Implementation of **chatbot and dialogue system experiments**, including:

* Prompt-based interactions
* Natural language understanding
* Conversational workflows
* Language model integration

### Big Data

Experiments involving **large-scale data processing and analytics**, including:

* Data preprocessing pipelines
* Distributed data analysis
* Data transformation and aggregation
